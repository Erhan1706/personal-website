Quizzz

Quizzz was developed during my first year degree at TU Delft. It was developed for a class named "OOP Project" and it was a group project made by me and 5 other students. It was my first large scale project, and it help me learn a lot of important tools, like version control and relational databases.

On our modern consumerist society, we constantly chase confort by spending without much though, because of this we are damaging the environment more than ever.

Thus, this game is a quiz game focused on raising awareness for our daily energy consumptions. The game can be played either in singleplayer or multiplayer, and it is made entirely in Java, using JavaFX for the GUI.

The server side uses REST API calls for communication which is implemented with Spring Boot.

It has lots of features, including different question formats (multiple choice/short answer), powerups, leaderboards, game costumization, an endless mode, etc...

Users can also contribute the the database of potential question, essential using CRUD operations in a H2 database.
